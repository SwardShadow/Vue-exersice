## 实现点击加载更多评论的功能

1. 为加载更多按钮绑定点击事件，在事件中请求下一页数据
1. 点击加载更多，让`pageIndex++`,然后重新调用this.getComments()方法重新获取最新一页的数据
1. 为了防止新数据覆盖老数据的情况，我们在点击加载更多的时候，每当获取到新数据，应该让老数据调用数组的concat方法拼接上新数组
